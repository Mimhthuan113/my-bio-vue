# 🤝 Contributing Guide

Cảm ơn bạn đã quan tâm đến việc đóng góp cho dự án Portfolio! Dưới đây là hướng dẫn để bắt đầu.

## 1. Quy trình đóng góp

1. **Fork** repository này.
2. Tạo một **Branch** mới cho tính năng hoặc bản sửa lỗi của bạn (`git checkout -b feat/tinh-nang-moi`).
3. Thực hiện các thay đổi và **Commit** theo chuẩn [Coding Conventions](./CODING_CONVENTIONS.md).
4. **Push** branch của bạn lên fork (`git push origin feat/tinh-nang-moi`).
5. Tạo một **Pull Request (PR)** về repo chính.

## 2. Tiêu chuẩn Pull Request

- PR nên tập trung vào một vấn đề duy nhất.
- Đảm bảo code đã được format bằng `npm run lint`.
- Cần có mô tả rõ ràng về những gì bạn đã thay đổi hoặc thêm vào.
- Đính kèm screenshot hoặc video nếu có thay đổi về UI.

## 3. Báo cáo lỗi (Bug Reports)

Vui lòng sử dụng **GitHub Issues** để báo cáo lỗi. Cung cấp càng nhiều thông tin càng tốt (OS, Browser, Các bước tái hiện lỗi).
